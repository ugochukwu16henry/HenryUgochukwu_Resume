# Henry M. Ugochukwu - Full Stack Developer Resume

A modern, professional digital resume with print and PDF export functionality.

## Files Structure

- `index.html` - Main resume page (required for GitHub Pages)
- `resume.css` - Stylesheet
- `images/Henrypicture.jpg` - Profile picture
- `.nojekyll` - Prevents Jekyll processing on GitHub Pages

## GitHub Pages Deployment

To deploy this resume on GitHub Pages:

1. **Push all files to your GitHub repository**
   - Make sure `index.html` is in the root directory
   - Include `resume.css`, `images/` folder, and `.nojekyll` file

2. **Enable GitHub Pages:**
   - Go to your repository Settings
   - Navigate to "Pages" in the left sidebar
   - Under "Source", select the branch (usually `main` or `master`)
   - Select the folder (usually `/root`)
   - Click "Save"

3. **Wait for deployment:**
   - GitHub Pages usually takes 1-2 minutes to deploy
   - Your site will be available at: `https://[username].github.io/[repository-name]`

4. **Verify file paths:**
   - Ensure all file paths are correct and case-sensitive
   - The image path should be: `images/Henrypicture.jpg`
   - The CSS path should be: `resume.css`

## Local Testing

Open `index.html` in your web browser to view the resume locally.
